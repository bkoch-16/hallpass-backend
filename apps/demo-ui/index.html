Hallpass API Demo
About this demo ▴

This is a portfolio project demonstrating a backend REST API. There is no full frontend, this tool lets you explore the API endpoints directly, inspect responses, and interact with the data layer.

Getting started: Select a stage (Prod is more stable), then go to Auth > Sign In and use one of the seed accounts below. The session cookie is then used automatically by all protected endpoints.

Seed accounts (password: password): student@hallpass.dev   teacher@hallpass.dev   admin@hallpass.dev   superadmin@hallpass.dev

  • Dev environment is actively used for development and testing and may be less stable than Prod.
  • Prod credentials are not normally shared publicly, but are provided here to offer a more stable demo environment.
  • Cold starts may cause the first request to take a few seconds. In a real production setting I would set minimum instances to 1 or more to prevent this startup time, but I keep them at 0 in a portfolio to keep infrastructure costs minimal.
